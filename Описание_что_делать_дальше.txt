ФИНАЛЬНАЯ ВЕРСИЯ ПРОЕКТА

Что изменилось:
1. Теперь проект прогнозирует не долю несовершеннолетних правонарушителей, а индекс риска цифровой девиации digital_deviation_risk_score от 0 до 100.
2. В обучающую выборку добавлены синтетические сценарии низкого, среднего и высокого риска.
3. Благодаря этому модель стала логичнее: низкие цифровые факторы дают низкий результат, средние — средний, высокие — высокий.

Что делать дальше:

1. Открой папку проекта в VS Code или PyCharm.
2. Установи зависимости:
   python -m pip install -r requirements.txt

3. Запусти обучение заново:
   python run_pipeline.py

4. Запусти API:
   python -m uvicorn main:app --reload

5. Открой Swagger:
   http://127.0.0.1:8000/docs

6. Выбери POST /deviant-forecast.
7. Нажми Try it out.
8. Вставь JSON из одного из файлов:
   - test_api_request_low.json
   - test_api_request_medium.json
   - test_api_request_high.json

Ожидаемые результаты:
- low: примерно 18, уровень низкий;
- medium: примерно 46, уровень средний;
- high: примерно 87, уровень высокий.

Как объяснять преподавателю:

Итоговая модель прогнозирует интегральный индекс риска цифровой подростковой девиации. Индекс рассчитывается на основе цифровых факторов: интернет-активности, использования социальных сетей, экранного времени, интернет-зависимости и стресса от цифровых устройств. Для корректной работы модели исходная выборка была расширена синтетическими сценариями, отражающими низкий, средний и высокий уровень цифрового риска.
