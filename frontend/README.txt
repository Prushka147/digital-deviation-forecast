Сайт-анкета для проекта Digital Deviation Forecast API

Как запустить:
1) Запусти свой FastAPI проект:
   python -m uvicorn main:app --reload
2) Открой файл index.html в браузере.

Как работает:
- Пользователь проходит анкету.
- Ответы преобразуются в признаки модели.
- Сайт отправляет POST-запрос на http://127.0.0.1:8000/deviant-forecast
- Если API не запущен, сайт использует локальный демонстрационный расчет.

Файлы:
index.html - разметка сайта
style.css - дизайн
script.js - логика анкеты, отправка в API, графики
